# Arlott

A personal GitHub Pages site built with [VitePress](https://vitepress.dev/).

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/)
- [pnpm](https://pnpm.io/)

### Installation

```bash
pnpm install
```

### Development

Start the local development server:

```bash
pnpm docs:dev
```

### Build

Build the site for production:

```bash
pnpm docs:build
```

### Preview

Preview the production build locally:

```bash
pnpm docs:preview
```

## Deployment

The site is automatically deployed to [GitHub Pages](https://arlott.github.io/) on push.
